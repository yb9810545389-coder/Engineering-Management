### **Unit IV: Human Resource Management and Control (8 Hrs.)**

**Core Concept:**
Sir/Mam, Engineering management vaneko project matra manage garne haina. Project banaune **Manchhe (HR)** lai manage garne ra project **Track (Control)** ma chha ki chhaina vanera check garne unit ho yo.

---

### **4.1 Human Resource Management (HRM)**

**Definition:**
HRM vaneko organization ma **Right Manchhe** lai **Right Time** ma **Right Place** ma lyaune, train garne, ra tikairakhne process ho.
*   *Neplish Logic:* "Manchhe lyaune (Recruit), sikaune (Train), ani kaam gareko cha ki chaina herne (Appraise)."

#### **4.1.2 Job Analysis (The Foundation)**

Vacancy kholnu vanda aghi hami le **Job Analysis** garnu parcha. Yo vaneko tyo specific post ko lagi k kasto manchhe chaincha vanera study garne ho. Yesko duita output huncha (Viva ma sure sodhcha):

1.  **Job Description (JD):**
    *   Yo **Position/Kaam** ko barema ho.
    *   *Includes:* Job title, Duties, Responsibilities, Location.
    *   *Example:* "Daily 8 hours coding garnu parcha, bugs fix garnu parcha."
2.  **Job Specification (JS):**
    *   Yo **Manchhe (Candidate)** ko barema ho.
    *   *Includes:* Qualification, Skills, Experience.
    *   *Example:* "BE Computer pass gareko hunu paryo, Java ra Python aaunu paryo."

---

#### **4.1.3 Recruitment and Selection (Differences)**

Yo topic viva ma external le confuse garna sodhcha.

1.  **Recruitment (Bhartaa):**
    *   Yo **Positive Process** ho.
    *   *Goal:* Dherai vanda dherai manchhe le apply garun. (Crowd jamma garne).
    *   *Example:* Facebook/LinkedIn ma vacancy post garne.
2.  **Selection (Chhanot):**
    *   Yo **Negative Process** ho.
    *   *Goal:* Unqualified manchhe lai **Reject** garne ra best lai matra rakhne.
    *   *Steps:* Screening $\to$ Interview $\to$ Testing $\to$ Hiring.

---

#### **4.1.4 Human Resource Training (Sikaune)**

Employee hire garepachi uslai kaam sikaunu parcha.
1.  **On-the-Job Training:**
    *   *Concept:* Kaam gardai sikne. "Learning by doing."
    *   *Methods:* Internship, Apprenticeship, Mentoring.
    *   *Faida:* Real work experience huncha, cost kam lagcha.
2.  **Off-the-Job Training:**
    *   *Concept:* Kaam bata chhutti liyera class ma sikne.
    *   *Methods:* Workshops, Seminars, Online Courses (Udemy/Coursera), Case Studies.
    *   *Faida:* Distraction hudaina, theory ramro sanga bujhincha.

---

#### **4.1.5 Performance Appraisal (Mulyankan)**

**Concept:**
Employee le ramro kaam garyo ki naramro? Salary badaune ki fire garne? Yo decision lina **Appraisal** garincha.

**Common Methods:**
*   **360-Degree Feedback:** Employee ko barema boss le matra haina, peers (sathi), juniors, ra clients sabai le feedback dinchan.
*   **MBO (Management by Objectives):** Boss ra Employee mile ra goal set garne. Last ma tyo goal pura vayo ki vayena check garne.

---

#### **4.1.6 Challenges in Managing ICT Workforce**

IT/Engineering field ma manchhe manage garna garo chha. Kina?
1.  **Rapid Skill Obsolescence:** Aaja sikya technology 2 barsa pachi purano huncha. (Continuous training chaincha).
2.  **High Turnover:** Ramro developer lai aru company le badi paisa diyera chado lagdincha.
3.  **Burnout:** Deadline ra bug fixing ko pressure le manchhe chado thakchan.
4.  **Remote Work Management:** Team member world ko farak farak time zone ma huda communication gap huncha.

---

### **4.2 Control (Niyantran)**

**Definition:**
Plan anusar kaam vayeko chha ki chhaina vanera check garne ra galti vaye sachaune process ho. "Planning is looking ahead, Controlling is looking back."

#### **4.2.2 Process of Control (4 Steps)**

Yo step viva ma serial order ma vannu parcha:
1.  **Establish Standards:** Target set garne. (Ex: "Yo project 10 din ma sakne").
2.  **Measure Performance:** Aaja samma kati kaam vayo napne. (Ex: "Aaja 5th day ho, tara 30% matra kaam vayo").
3.  **Compare:** Target vs Actual daajne. (Ex: "Hami 20% le pachadi chhau").
4.  **Corrective Action:** Galti sudharne. (Ex: "Overtime kaam garne" wa "Thap manchhe rakhne").

#### **4.2.4 ICT Tools for Effective Control**

Engineering project lai haat le lekhera control garna sakidaina. Hami lai tools chaincha:
1.  **Microsoft Project (MS Project):**
    *   Planning, Scheduling, ra Gantt Chart banauna use huncha. Project kahile sakkincha visual rup ma herna milcha.
2.  **Jira:**
    *   Vishesh gari Software Engineering ma **Bug Tracking** ra **Agile/Scrum** management ko lagi use huncha.
3.  **Trello / Asana:**
    *   Task management tool. (To Do, Doing, Done list banauna).
4.  **Version Control (Git/GitHub):**
    *   Code ko changes track garna ra team collaboration ko lagi.
